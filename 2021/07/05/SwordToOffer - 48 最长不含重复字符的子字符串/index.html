《剑指Offer》 48 - 最长不含重复字符的子字符串

题目：剑指 Offer 48. 最长不含重复字符的子字符串

请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。

此题与LeetCode3相同：3. 无重复字符的最长子串

题解：

一维动态规划问题：

状态定义：设动态规划列表 $dp$，$dp[i]$ 代表以字符 $s[i]$ 为结尾的最长不含重复字符串的子字符串长度

状态转移方程： 固定 $i$，从 $i$ 往 前找与 $s[i]$ 相同的字符 $s[j]$，距离为 $i - j$

  • 没有找到 s[j] , 且已经找到了字符串开头，则 $dp[i] = dp[i-1] + 1$
  • $dp[i-1] < i - j$ ，说明与$s[i]$ 相同的字符 $s[j]$ 在上个 $dp[i-1]$的区间之外，则 $dp[i] = dp[i-1] + 1$
  • $dp[i-1] \ge i - j$ ，说明与$s[i]$ 相同的字符 $s[j]$ 在上个 $dp[i-1]$的区间之内，则 $dp[i] = i - j$

返回值：$max(dp)$

注意事项：做动态规划题目前，写完状态转移方程后，想一想能否用常数个变量完成动态规划过程，从而避免开辟整个 $dp$ 数组，节省空间开销。

方法一：动态规划 + 向前遍历寻找 $s[j]$ ， $O(N^2) - O(1)$

方法二：动态规划 + 哈希表 $O(N) - O(1)$

字符的 ASCII 码范围为 0 ~ 127，哈希表 dic 最多使用 $O(128) = O(1)$大小的额外空间，所以空间复杂度为 $O(1)$，哈希表存放着最近一次出现字符的所在位置。

举例说明代码中第一次出现某字符 j = -1 的原因，例如字符串”aba”：

  • 当i = 0 时，mp还是空的，当然找不到 "a"，所以j = -1，后面 i - j = 0 - (- 1) = 1，dp_temp在 i = 0 处等于 1，res = 1

  • 当 i = 1时，mp中只有"a"，找不到 "b"，所以 j仍然是-1，后面i - j = 1 - (-1) = 2，dp_temp在 i = 1 处等于2，res = 2

  • 当 i = 2 时，mp中已经有了"a"，mp["a"] = 1，j = 1，然后更新一下，令当前的索引 i = 2 赋值给 mp["a"]，这样mp["a"]就记录当前 "a"的出现位置，j 记录了上一个 "a" 出现的位置，i - j = 2 - 1 = 1 < res，dp_temp在i = 2处等于1，res = 2

方法三：哈希表 + 滑动窗口 $O(N) - O(1)$

  • 哈希表记录字符s[i] 最后一次出现的索引
  • 每轮更新左边界 j，保证区间 [j + 1, i] 内无重复字符且最大
  • 每轮更新结果res，取上一轮的res与本轮双指针区间 [j + 1, i]的宽度 i - j比较，取最大值
